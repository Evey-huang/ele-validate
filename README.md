# ele-validate

> element-ui表单验证，内置的验证规则太少了，日常工作不够用，以这个为基础封装了验证规则。

## Usage

``` bash
# clone repo
git clone https://github.com/Evey-huang/ele-validate.git

# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

```
